## Reverting Hexdump
Step 1: Confirm the file is a hexdump
#### Example hexdump data
```
$ cat hexdump_data | head
```

To view data in hexump use `xxd filename`

Step 2: Reverse hexdump to data
`xxd -r hexdump_data compressed_data`

Step 3: `file compressed_data`
https://en.wikipedia.org/wiki/List_of_file_signatures

## Decompressing files
```
gzip -d filename
bzip2 -d filename
tar -xf filename
```
